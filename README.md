## 📖 Usage

### Creating a Room

1. Launch ASTRID
2. Click **"+ Create Room"**
3. Share the room ID with friends
4. Wait for others to join

### Joining a Room

1. Launch ASTRID
2. Click **"Enter ID room"**
3. Paste the room ID
4. Click **"Go in"**

### Room Features

| Action | Description |
|--------|-------------|
| 👑 **Leader** | First person in room, controls playback |
| 📋 **Copy ID** | Click room ID to copy to clipboard |
| 💬 **Chat** | Send messages in real-time |
| 🔄 **Sync** | Click sync button to resync video |
| 🚪 **Leave** | Exit button with confirmation |

### Leadership Transfer

- When the leader leaves, leadership passes to the person who joined earliest
- Leaders are marked with a 👑 crown icon
