## 📖 Usage

### Creating a Room

1. Launch ASTRID
2. Click **"+ Create Room"**
3. Share the room ID with friends
4. Wait for others to join

### Joining a Room

1. Launch ASTRID
2. Click **"Enter ID room"**
3. Paste the room ID
4. Click **"Go in"**
